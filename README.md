# cv-ui

UI concept for my sister's website. ([demo](https://www.dtjv.io/cv-ui/))

![website screen shot](./media/screen-shot.png)

## Development

`$ yarn install`

```sh
# in terminal 1
$ yarn watch:pug

# in terminal 2
$ yarn watch:css

# in terminal 3
$ yarn start
```

## Notes

`dist/` is included in repo because I keep static assets there.
